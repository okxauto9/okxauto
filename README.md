# OKX自动交易系统

一个基于OKX交易所API的自动化交易系统，支持合约交易，提供多种交易策略和风险管理功能。

## 功能特点

- 支持合约交易
- 多策略支持（网格交易、RSI策略）
- 自动风险管理（止盈止损、保证金管理）
- 实盘/模拟盘切换
- Web管理界面
- 完整的数据存储和分析

## 主要功能模块

### 1. 交易引擎
- 多策略支持
- 风险管理
- 自动保证金管理
- 止盈止损管理

### 2. API模块
- OKX API集成
- 请求频率控制
- 错误重试机制

### 3. 数据存储
- 交易记录
- 策略信号

- RESTful API
- JWT认证
- 实时监控
